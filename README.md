# Pakiip

**Pakiip** es una aplicación de entrega a domicilio (delivery) nativa para Android e iOS, construida utilizando **Flutter**.

## Descripción del Proyecto

Esta aplicación está diseñada para ofrecer servicios de entrega rápida de comida y compras de supermercado, todo en un solo lugar. La interfaz de usuario está en constante desarrollo para ofrecer una experiencia premium y fácil de usar.

## Características Principales

*   **Multiplataforma:** Disponible para Android e iOS.
*   **Interfaz Moderna:** Diseño atractivo y funcional.
*   **Entrega Rápida:** Optimizado para la gestión eficiente de pedidos.
